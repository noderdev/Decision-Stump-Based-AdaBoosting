Instructions to Run 
* Install Scikit-learn , matplotlib, panda and numpy.
* Unzip the zip file and open the folder in jupyter. 
* Open ab.ipynb file and run. 
* Once 200 rounds are run, the predicted csv file corresponding to each boosting round for test data is generated. The csv corresponding to single decision tree is also created. 
* Once the final graph is printed ie the test and train error , the Jupyter notebook has executed.
